# Mutli Vendor App

## requirements
- php > 8
- composer > 2.2
- mysql 8

## Database
Create database and a .env file from .env.example and modify the configuration
run the database.sql in your mysql management system.

## Install 
```
composer install
```
```
npm install
```
```
npm run build
```
```
php artisan key:generate
```


## Run 
```
php artisan serve
```
